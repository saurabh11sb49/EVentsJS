What is eventListener??

Event listeners in JavaScript are functions that wait for a specific event to occur on a particular DOM element. When that event happens, the function is executed, allowing you to create interactive web pages.

Events can be user interactions (clicks, key presses, mouse movements) or browser events (page load, resize, etc.). The basic syntax for adding an event listener is:

element.addEventListener(event, function);
